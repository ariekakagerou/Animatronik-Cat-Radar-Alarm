# Animatronik Cat Radar Alarm

Robot animatronik berbentuk kucing yang berfungsi sebagai sistem alarm
menggunakan sensor ultrasonik dan Arduino Uno.

## Fitur
- Deteksi objek menggunakan HC-SR04
- Servo bergerak 0°–180° sebagai radar
- Buzzer dan LED sebagai alarm

## Komponen
- Arduino Uno R3
- Sensor Ultrasonik HC-SR04
- Servo Motor SG90
- Buzzer
- LED

## Cara Menjalankan
1. Upload file `cat.ino` ke Arduino
2. Hubungkan ke power bank
3. Robot berjalan otomatis
